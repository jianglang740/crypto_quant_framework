**为什么用 `Decimal` 而不是 `float`？**

```python
# float 的问题
0.1 + 0.2  # 结果是 0.30000000000000004，不是精确的 0.3

# Decimal 的精确性
Decimal("0.1") + Decimal("0.2")  # 结果是精确的 Decimal("0.3")
```

金融计算中，一分钱的误差都可能导致严重问题，所以必须用 `Decimal`。

### returns() 方法（第83-85行）：计算收益率序列

```python
def returns(self, equity_curve: list[tuple[object, Decimal]]) -> list[float]:
    equities = [equity for _, equity in equity_curve]
    return [float(equities[i] / equities[i - 1] - Decimal("1")) for i in range(1, len(equities)) if equities[i - 1]]
```

**功能：** 计算每期的收益率，返回一个收益率列表。

**逐行解读：**

```python
equities = [equity for _, equity in equity_curve]  # 提取权益值
```

```python
return [float(equities[i] / equities[i - 1] - Decimal("1")) for i in range(1, len(equities)) if equities[i - 1]]
```

**列表推导式拆解：**

| 部分 | 含义 |
|------|------|
| `range(1, len(equities))` | 从索引1开始遍历到最后 |
| `equities[i] / equities[i - 1] - Decimal("1")` | 当期收益率 = 当期权益 / 上期权益 - 1 |
| `if equities[i - 1]` | 防止除以0 |
| `float(...)` | 转换为浮点数（后续计算需要） |

**例子：**

```python

# 计算过程
# i=1: 10100/10000 - 1 = 0.01
# i=2: 10050/10100 - 1 ≈ -0.00495
# i=3: 10200/10050 - 1 ≈ 0.01493

# 输出

```

### drawdowns() 方法（第87-96行）：计算回撤曲线

```python
def drawdowns(self, equity_curve: list[tuple[object, Decimal]]) -> list[tuple[object, Decimal]]:
    if not equity_curve:
        return []
    peak = equity_curve[0][1]
    result = []
    for timestamp, equity in equity_curve:
        peak = max(peak, equity)
        drawdown = equity / peak - Decimal("1") if peak else Decimal("0")
        result.append((timestamp, drawdown))
    return result
```

**功能：** 计算每个时间点的回撤率，返回回撤曲线。

**与 `_max_drawdown` 的区别：**

| 方法 | 返回值 | 用途 |
|------|--------|------|
| `_max_drawdown` | 单个数值（最大回撤） | 绩效报告中的一个指标 |
| `drawdowns` | 列表（每个时间点的回撤） | 绘制回撤曲线图 |

**逐行解读：**

```python
if not equity_curve:
    return []  # 空数据返回空列表
```

```python
peak = equity_curve[0][1]  # 初始峰值 = 第一个权益值
```

```python
result = []
for timestamp, equity in equity_curve:  # 遍历每个时间点
    peak = max(peak, equity)  # 更新峰值（只升不降）
    drawdown = equity / peak - Decimal("1") if peak else Decimal("0")  # 计算当前回撤
    result.append((timestamp, drawdown))  # 保存结果
return result
```

**例子：**

```python
# 输入
equity_curve = [
    ("2024-01-01", 10000),
    ("2024-01-02", 10500),
    ("2024-01-03", 10200),
    ("2024-01-04", 9800),
]

# 计算过程
# 第1天: peak=10000, drawdown=10000/10000-1=0
# 第2天: peak=10500, drawdown=10500/10500-1=0
# 第3天: peak=10500, drawdown=10200/10500-1≈-0.0286
# 第4天: peak=10500, drawdown=9800/10500-1≈-0.0667

# 输出
drawdowns = [
    ("2024-01-01", 0),
    ("2024-01-02", 0),
    ("2024-01-03", -0.0286),
    ("2024-01-04", -0.0667),
]
```

### ⚠️ 重点：realized_pnls() 方法（第98-126行）

这是最复杂的方法，负责计算每笔"已闭合交易"的盈亏。

```python
def realized_pnls(self, trades: list[Trade]) -> list[Decimal]:
    amount = Decimal("0")
    entry_price = Decimal("0")
    entry_fee = Decimal("0")
    realized_pnls: list[Decimal] = []
```

**状态变量：**

| 变量 | 初始值 | 含义 |
|------|--------|------|
| `amount` | 0 | 当前持仓量（正数=多头，负数=空头） |
| `entry_price` | 0 | 当前持仓的平均成本价 |
| `entry_fee` | 0 | 当前持仓累计的手续费 |
| `realized_pnls` | [] | 已实现盈亏列表 |

---

#### 主循环（第103-125行）

```python
    for trade in trades:
        signed_amount = trade.amount if trade.side.value == "buy" else -trade.amount
```

**有符号数量：**

- 买入 → 正数（增加持仓）
- 卖出 → 负数（减少持仓）

---

#### 情况1：同方向开仓（第105-110行）

```python
        if amount == 0 or (amount > 0) == (signed_amount > 0):
            total_amount = abs(amount) + abs(signed_amount)
            entry_price = (entry_price * abs(amount) + trade.price * abs(signed_amount)) / total_amount
            entry_fee += trade.fee
            amount += signed_amount
            continue
```

**条件解读：**

| 条件 | 含义 |
|------|------|
| `amount == 0` | 当前没有持仓 |
| `(amount > 0) == (signed_amount > 0)` | 当前持仓方向和新交易方向一致 |

**满足条件时：**

1. 累加持仓量
2. 重新计算平均成本价
3. 累加手续费
4. 跳过后续代码（`continue`）

**平均成本价公式：**

```
新均价 = (原持仓成本 + 新交易成本) / 新总持仓量
```

例子：
- 原持仓：0.5 BTC @ 10000，成本 = 5000
- 新买入：0.5 BTC @ 11000，成本 = 5500
- 新均价 = (5000 + 5500) / 1.0 = 10500

---

#### 情况2：反向平仓（第111-125行）

```python
        close_amount = min(abs(amount), abs(signed_amount))
        fee_share = entry_fee * close_amount / abs(amount)
        if amount > 0:
            pnl = (trade.price - entry_price) * close_amount - fee_share - trade.fee
        else:
            pnl = (entry_price - trade.price) * close_amount - fee_share - trade.fee
        realized_pnls.append(pnl)
```

**平仓逻辑：**

1. `close_amount`：本次平仓数量
2. `fee_share`：分摊的手续费
3. 计算盈亏：
   - 多头平仓：`(卖出价 - 成本价) * 数量 - 手续费`
   - 空头平仓：`(成本价 - 买入价) * 数量 - 手续费`
4. 记录盈亏

---

#### 更新剩余持仓（第118-125行）

```python
        remaining_amount = abs(amount) - close_amount
        if remaining_amount == 0:
            amount = Decimal("0")
            entry_price = Decimal("0")
            entry_fee = Decimal("0")
        else:
            amount += signed_amount
            entry_fee -= fee_share
```

**两种情况：**

| 情况 | 处理 |
|------|------|
| 完全平仓 | 清空所有状态变量 |
| 部分平仓 | 更新剩余持仓和手续费 |

```python
        final_equity=Decimal("0"),
        # ... 所有字段都是0
    )
```

**用途：** 当没有数据时，返回一个所有指标都是0的报告，避免返回 `None` 导致后续代码报错。

### _annual_return() 方法（第151-154行）：年化收益率

```python
def _annual_return(self, total_return: Decimal, periods: int) -> Decimal:
    if periods <= 1:
        return Decimal("0")
    return Decimal(str((1 + float(total_return)) ** (self.periods_per_year / (periods - 1)) - 1))
```

**公式：**

```
年化收益率 = (1 + 总收益率) ^ (一年周期数 / 回测周期数) - 1
```

**例子：**

```python
# 回测100天，总收益率10%
total_return = 0.1
periods = 100
periods_per_year = 365

annual_return = (1 + 0.1) ^ (365 / 99) - 1
              = 1.1 ^ 3.687 - 1
              ≈ 0.42（42%）
```

**为什么是 `periods - 1`？**

- `periods` 是权益点的数量
- 收益率区间数量 = `periods - 1`
- 例如：3个权益点 → 2个收益率区间

### _max_drawdown() 方法（第156-164行）：最大回撤

```python
def _max_drawdown(self, equities: list[Decimal]) -> Decimal:
    peak = equities[0]
    max_drawdown = Decimal("0")
    for equity in equities:
        peak = max(peak, equity)
        if peak:
            drawdown = equity / peak - Decimal("1")
            max_drawdown = min(max_drawdown, drawdown)
    return max_drawdown
```

**核心逻辑：**

1. 维护一个"只升不降"的峰值 `peak`
2. 每个时间点计算：`当前回撤 = 当前权益 / 历史峰值 - 1`
3. 记录最小的回撤（最严重的亏损）

**例子：**

```python
equities = [10000, 10500, 10200, 9800, 11000]

# 遍历过程
# 10000: peak=10000, drawdown=0, max_drawdown=0
# 10500: peak=10500, drawdown=0, max_drawdown=0
# 10200: peak=10500, drawdown=-0.0286, max_drawdown=-0.0286
# 9800:  peak=10500, drawdown=-0.0667, max_drawdown=-0.0667
# 11000: peak=11000, drawdown=0, max_drawdown=-0.0667

# 结果
max_drawdown = -0.0667（-6.67%）
```

### _volatility() 方法（第166-171行）：年化波动率

```python
def _volatility(self, returns: list[float]) -> float:
    if len(returns) < 2:
        return 0.0
    mean_return = sum(returns) / len(returns)
    variance = sum((item - mean_return) ** 2 for item in returns) / (len(returns) - 1)
    return sqrt(variance) * sqrt(self.periods_per_year)
```

**公式：**

```
波动率 = 标准差 × √一年周期数
```

**步骤：**

1. 计算平均收益率
2. 计算方差（样本方差，除以 n-1）
3. 开方得到标准差
4. 年化：乘以 √365

**为什么除以 `n-1`？**

这是**样本标准差**，统计学中的标准做法，用于估计总体标准差。

### _sharpe_ratio() 方法（第173-181行）：夏普比率

```python
def _sharpe_ratio(self, returns: list[float]) -> float:
    if len(returns) < 2:
        return 0.0
    mean_return = sum(returns) / len(returns)
    variance = sum((item - mean_return) ** 2 for item in returns) / (len(returns) - 1)
    volatility = sqrt(variance)
    if volatility == 0:
        return 0.0
    return mean_return / volatility * sqrt(self.periods_per_year)
```

**公式：**

```
夏普比率 = 平均收益率 / 波动率 × √一年周期数
```

**含义：** 每承担一单位风险，能获得多少超额收益。

**解读：**

| 夏普比率 | 评价 |
|----------|------|
| < 1 | 较差 |
| 1 - 2 | 良好 |
| > 2 | 优秀 |

### _sortino_ratio() 方法（第183-192行）：索提诺比率

```python
def _sortino_ratio(self, returns: list[float]) -> float:
    downside_returns = [item for item in returns if item < 0]
    if len(downside_returns) < 2:
        return 0.0
    mean_return = sum(returns) / len(returns) if returns else 0.0
    downside_variance = sum(item**2 for item in downside_returns) / (len(downside_returns) - 1)
    downside_volatility = sqrt(downside_variance)
    if downside_volatility == 0:
        return 0.0
    return mean_return / downside_volatility * sqrt(self.periods_per_year)
```

**与夏普比率的区别：**

| 指标 | 风险度量 |
|------|----------|
| 夏普比率 | 全部收益率的标准差 |
| 索提诺比率 | 只有负收益率的标准差 |

**为什么只看负收益？**

- 投资者只关心"亏钱的风险"
- 正收益的波动不是风险，而是机会
